# ADR 0002: Linguagem de Programacao

## Status
Aceito

## Contexto
O projeto permite escolha entre Java 21 ou Kotlin 2.1.x.

## Decisao
Utilizar Java 21 como linguagem principal.

## Justificativa
- Linguagem padrao do ecossistema Spring
- Amplo suporte de bibliotecas e frameworks
- Facilidade de onboarding para equipes Java
- Melhor suporte a ferramentas de build e analise
- Compatibilidade total com Java 21

## Consequencias
- Positivas:
  - Maior previsibilidade no time
  - Compatibilidade direta com bibliotecas Java
  - Menor risco de integracao
- Negativas:
  - Mais boilerplate que Kotlin
